\section{Deliverables}
\label{sec:deliverables}

The following deliverables are available for this project:

\begin{itemize}
    \item \textbf{GitHub Repository:} The complete source code, documentation, and project files are available at \url{https://github.com/rp-bot/ableton-buddy}
    
    \item \textbf{Demo Video:} A demonstration video showcasing the system's capabilities is included in the repository, demonstrating the natural language interface controlling Ableton Live in real-time.
    
    \item \textbf{Documentation:} The documentation for the system is available in the repository, including the report, and other relevant files.
\end{itemize}
